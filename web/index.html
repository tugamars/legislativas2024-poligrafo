Informações

Funcionamento

Fact-checks recolhidos/analisados a começar no dia 5 fevereiro 2024. São analisados todos os fact-checks envolvendo os candidatos na lista. Todos os fact-checks analisados são do poligrafo.sapo.pt.

Pontuação

A pontuação é atribuída da seguinte forma:

  • Verdadeiro: 0 pontos
  • Verdadeiro, mas...: -1 ponto
  • Impreciso: -2 pontos
  • Descontextualizado: -3 pontos
  • Falso: -4 pontos
  • Manipulado: -5 pontos
Sendo o quadro ordenado pelo quoeficiente de pontos por artigo analisado. Sendo 0 ou o mais próximo de 0 a melhor pontuação que pode ser obtida.

Atualização

Os dados não são atualizados automaticamente, sendo que a data de última atualização consta da tabela.

Algoritmo

O algorítmo utilizado tanto para recolha como para análise dos dados encontra-se disponível no github!
